交互原则

  • hover 仅改变背景色（极浅灰 #F8FAFC 或 #F1F5F9），150ms ease
  • 无 box-shadow（任何元素）
  • 无 transform: scale()（任何元素）
  • 无 transform: translateY()（hover 效果）
  • 无渐变背景（按钮、卡片等）
  • Focus 状态: border-color 变为 #0F172A，outline: none
  • 所有 transition: all 150ms ease
